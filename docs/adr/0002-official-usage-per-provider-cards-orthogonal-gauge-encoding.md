# 공식 사용량을 provider별 카드로 분리하고, 게이지 인코딩을 직교화한다(색=임계·텍스처=공식/추정)

단일 "공식 사용량" 카드 하나에 Claude 섹션과 Codex 섹션을 텍스트로 쌓아 올리던 구조는 두 가지 문제가 있었다. (1) provider마다 공식 API가 **독립적으로 성공/실패**하는데(Claude는 취득됐고 Codex는 토큰 만료 같은) 한 카드 안에 묶여 상태 경계가 흐렸고, (2) 전부 텍스트라 "한눈에 안 들어오고", Gemini 등 provider를 더하면 한 카드가 감당 못 한다. 그래서 공식 사용량을 **provider별 카드 그리드**로 분리하고(가시성은 기존 `tracked_providers`가 게이트 — 카드 +1 = provider +1), 텍스트 나열을 **게이지 시각화**로 바꾼다.

핵심은 게이지가 동시에 전달해야 하는 두 정보 차원을 **직교 인코딩**으로 분리한 것이다:

- **색 = 임계 전용.** fill 색은 raw `utilization`로만 결정한다(녹 <75% · 앰버 75~90% · 적 ≥90%, 100% 도달은 '소진' 라벨). 모든 버킷·provider·단위(USD 한도/rate-window %/크레딧)에 동일 적용된다.
- **텍스처 = 출처(공식/추정) 전용.** 공식(ground truth) 버킷은 solid fill, 추정 버킷(Codex 주간 = 월÷4)은 해치 fill + 점선 트랙. 칩 라벨(공식/추정)은 보조.
- provider 정체성 색(Claude=코랄, Codex=상태색을 피한 별도 hue)은 **카드 크롬(좌측 액센트 바·이니셜)에만** 두고 게이지 fill엔 절대 쓰지 않는다.

이 직교화는 CONTEXT.md가 사활을 거는 **공식/추정 경계의 도메인 무결성**을 시각 규칙으로 못박는다. Codex 카드처럼 한 카드 안에 공식 게이지(월간)와 추정 게이지(주간)가 나란히 설 때, 추정값이 ground truth처럼 읽히면 안 되기 때문이다.

## 고려한 대안

- **출처를 색으로 구분(공식=진한 색, 추정=회색조)**: 거부. 색을 임계에도 쓰면 한 채널에 두 의미가 겹쳐 추정 게이지가 "위험(적색)"인지 "추정(회색)"인지 모호해지고, 지키려던 공식/추정 경계가 오히려 흐려진다.
- **칩 라벨만 유지(기존 방식)**: 거부. 흘끗 볼 때 fill 모양이 동일해 추정이 공식처럼 읽힌다. 애초 "글자만이라 한눈에 안 들어온다"는 피드백의 원인이기도 하다.
- **active 버킷 1개를 히어로 게이지로 크게 + 나머지 축소**: 거부. 한 카드의 버킷들은 만료·리셋·한도가 각자 다른 별개 자원이라(예: 포함된 크레딧은 2026-09-10 만료) 보조로 죽이면 그 정보가 묻힌다. 모든 버킷을 동등 게이지로 그린다.
- **≥100%를 적색 임계로**: 거부. provider 강제 한도는 100%가 사실상 상한이라 적색이 거의 안 켜지는 죽은 신호가 된다. 경계를 아래로 당겨 적색이 "한도 임박"을 뜻하게 했다.

## 결과

- 새 provider 추가 = `tracked_providers` +1 = 그리드 카드 +1. 카드 컴포넌트는 provider 무관하게 동일 규칙(utilization% fill + 단위별 캡션 + 출처 텍스처)을 따른다.
- fetch 실패(auth_error/http_error)해도 카드는 직전 스냅샷으로 **스탈 게이지 + ⚠경고 칩**을 그린다. 게이지를 죽이지 않는다.
- 공식 데이터가 전혀 없는 provider는 한도가 없어 게이지를 못 그린다 → **사용량 전용 view**(로컬 추정 숫자 + 미니 스파크라인, 한도/번다운 없음)로 격하한다.
- 예측 렌즈(소진 예상)는 게이지 위 **고스트 fill**(현재→리셋시 예상 위치)로 표현하되, 렌즈가 Claude active 버킷 전용이라 Claude 카드에만 뜬다 — provider-parity 약화를 interim 한계로 수용한다(Codex는 주간 추정 게이지가 전방 신호 역할).
- 색=임계, 텍스처=출처라는 인코딩 규칙은 향후 추가되는 모든 게이지/카드가 따라야 하는 UI 컨벤션이 된다. 규칙 자체는 `static/src/input.css` 주석에 못박는다.
